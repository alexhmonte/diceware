Diceware

Diceware é um método utilizado para gerar senhas fortes que são fáceis de lembrar. O método basea-se no princípio de seleção aleatória de palavras a partir de uma lista de palavras, o que gera senhas resistentes a ataque de força bruta. O método tradicional Diceware pede a utilização de dados como fonte de números aleatórios. Esta aplicação utiliza um gerador de números aleatórios criptograficamente fortes no lugar de dados. Senha com seis ou mais palavras é segura o suficiente para uso em ambiente que necessitam de alta segurança. Senhas com cinco palavras são fortes o suficiente para serem utilizados como senha de acesso onde haja certos mecanismos de controle, como número limitado de tentativas, bloqueio temporário de acesso ou tempo de vida para a senha. Adicionalmente, é possível incluir alguns elementos como números, símbolos, letras maíusculas e definir um tamanho máximo para a senha para o cumprimento de regras para a aceitação de senhas.
